# Система тестирования

Консольное приложение для проведения тестирования на основе файлов с вопросами в специальном формате.

## Возможности

- 📖 Чтение тестов из `.txt` и `.docx` файлов
- 🎲 Случайная выборка вопросов
- 🔀 Перемешивание вариантов ответов
- 📊 Подсчет результатов и оценка
- 📝 Разбор ошибок с правильными ответами
- 🎨 Удобный консольный интерфейс
